# Daily Calorie Coach & Meal Planner

> A Java-based console application that calculates daily calorie needs and creates a meal plan based on user information and health goals.

## Overview

Daily Calorie Coach & Meal Planner is a group project developed as part of a Computer Science course. The application collects basic user information, calculates daily calorie needs, and suggests a meal schedule that fits the user's calorie target.

## Key Features

- Collects user information including age, weight, height, gender, and activity level
- Calculates daily calorie needs
- Generates a daily meal plan
- Includes breakfast, lunch, dinner, snacks, and dessert
- Displays meal information such as calories, protein, carbohydrates, and fats
- Provides Client and Admin modes
- Allows admins to add or update meals

## Built With

- Java
- Object-Oriented Programming
